# How to use
|Hotkey|Function|
|----|----|
|Home|show/hide tips|
|F1|Increase 1% support rate|
|F2|Decrease 1% support rate|
|F3|Increase 1000000 funds|
|F4|Decrease 1000000 funds|
|F5|Increase 1% chaos rate|
|F6|Decrease 1% chaos rate|
